## SMILEos 2.0 in HTML

Pretty much vibecoded for the parts I couldn't be bothered doing, other than that mostly just textures pulled from the [Ultrakill Wiki](https://ultrakill.wiki.gg/wiki/Category:SmileOS#) or made by me.

You're pretty much free to use anything here, as long as it isn't a custom texture (watch out in the `assets/` folder)

Useful for making Homepages and Social Media sharepoints like [StrawPage](https://straw.page/) does, like I've done here

Have fun!
